# Firmware

## How to Use

<Callout>
    You can test these command with arduino serial moniter.
</Callout>

- Check Status

```code copy
{"cmd": "status"}
```

- Check Version

```code copy
{"cmd": "version"}
```

- Reset Arduino

```code copy
{"cmd": "reset"}
```

- Make Arduino Status to Ready

```code copy
{"cmd": "setReady", "value": true}
```

- Check Arduino is Ready

```code copy
{"cmd": "isReady"}
```
