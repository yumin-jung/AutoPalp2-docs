# Client(Unity)

## Quick Start

<Steps>

### Clone git repository

```bash copy
git clone git@github.com:makelab-kaist/AutoPalp2.git
```

### Open **Unity Hub** and Add cloned repository

![add-repository](/img/autopalp2/add-repository.png)

### Open MainScene

- Assets > Scenes > MainScene

![open-main-scene](/img/autopalp2/open-main-scene.png)

### Build MainScene to Meta Quest 3

- File > Build Settings

![build-setting](/img/autopalp2/build-setting.png)

### Change Run Device to Meta Quest 3

- Android > Switch Platform > Change **Run Device** to Oculus Quest 3 > Build And Run

![switch-run-device](/img/autopalp2/switch-run-device.png)

</Steps>

## Manage Palpation Sequence

You can manage palpation step in Unity.

1. MainScene > Utils > PalpationSequenceManager
2. Manage order of `Step Objects` (Element 0 is the first region of palpation)

![palpation-step](/img/autopalp2/palpation-step.png)
